# 定价策略 v1.0

> 基于 10 平台 RPA 发布器的价值主张和用户分层

---

## 1. 分层模型

| 版本 | 价格 | 目标用户 | 平台数 | 核心差异 |
|------|------|---------|--------|---------|
| **免费版** | ¥0 | 个人创作者 | 3 | 微信 + 任意自选 2 平台 |
| **Pro 版** | ¥39/月 或 ¥299/年 | 自媒体运营 | 10 | 全部平台 + 定时发布 + 批量发布 |
| **企业版** | ¥199/月 | 团队/机构 | 10 | Pro 全部功能 + 多账号 + API + 优先支持 |

---

## 2. 功能边界

### 免费版
- 3 个平台（微信 + 任意 2 个）
- 单篇发布
- 基础编辑器
- 每日最多 5 篇
- 本地账号存储

### Pro 版
- 全部 10 平台
- 批量发布（一键多平台）
- 定时发布
- 视频上传
- 发布历史 + 统计看板
- 无限篇数
- 优先更新

### 企业版
- Pro 全部功能
- 多账号（每平台 5+ 个账号）
- API 接口（PROJECT-001 集成）
- 团队协作（共享账号池）
- SLA 支持（24h 响应）
- 定制化开发

---

## 3. 付费转化路径

```
免费体验 → 3 平台限制触发 → 升级弹窗 → 7 天 Pro 试用 → 付费订阅
```

**关键转化节点：**
1. 第 3 次发布时提示"已达到免费版每日上限"
2. 第 2 次选择第 4 个平台时提示"升级到 Pro 解锁所有平台"
3. 定时发布功能仅 Pro 可用（吸引重度用户）
4. 首次使用后 72h 弹窗 Pro 7 天免费试用

---

## 4. 支付方式

- 支付宝 / 微信支付（国内）
- Stripe（海外，后续）
- 兑换码（早期用户赠送）

---

## 5. 冷启动定价策略

**早期用户政策（前 200 名）:**
- Pro 年付 ¥99（原价 ¥299）— 终身锁定
- 企业版首月免费
- 推荐 3 人免费得 1 个月 Pro

**定价原则：**
- 比竞品（蚁小二 ¥49/月、Newrank ¥299/月）低 20-50%
- 以"10 平台 vs 竞品 5-7 平台"作为核心卖点
- 年付折扣 37%（¥299 vs ¥468 月付）
