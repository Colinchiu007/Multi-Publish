## 3. 转化漏斗

```
曝光（10,000） → 访问（1,000） → 注册下载（200） → 首次发布（80） → 留存（30） → 付费（8）
```

| 阶段 | 转化率 | 目标 |
|------|--------|------|
| 曝光 → 下载 | 2% | 优化 Landing Page |
| 下载 → 首次发布 | 40% | 简化 onboarding |
| 首次发布 → 次日留存 | 37% | 引导创建第二个平台账号 |
| 留存 → 付费 | 10% | 限时优惠弹窗 |

**关键优化：**
- 下载 → 首次发布 是最大流失点 — 减少首次启动时间，预置演示视频
- 次日留存 — 首次发布后推送"定时发布"功能引导
