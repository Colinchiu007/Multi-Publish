# Changelog

## v0.1.0 (2026-06-09)

初始版本。Electron 多平台内容一键发布桌面工具。

**支持的平台：**
- 微信公众号（草稿 → 群发）
- 知乎（文章+话题标签）
- 微博（长文发布）
- 抖音（图文发布）
- 小红书（标题+正文+标签）

**特性：**
- 富文本编辑器（Quill）
- 任务队列（顺序执行 + 自动重试）
- 多账号管理（Cookie 加密持久化）
- 定时发布（持久化 + 重启恢复）
- 发布历史 + 统计看板
- PROJECT-001 内容聚合器集成
- GitHub Actions CI 双平台构建
- 自动更新（electron-updater）
