## 发现的问题

### 浏览器模式下的预期警告（非 Bug）

1. **模型设置页面**: 显示 "electronAPI not available" 红色提示
   - 原因: 浏览器环境无 `window.electronAPI`
   - 影响: 仅浏览器截图时出现，Electron 内正常
   - 处理: 无需修复

2. **视频创作页面**: 显示 "Remotion 渲染引擎未就绪" 黄色警告
   - 原因: 浏览器环境无法调用 Electron IPC
   - 影响: 仅浏览器截图时出现，Electron 内正常
   - 处理: 无需修复
