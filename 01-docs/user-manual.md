# Multi-Publish 快速上手手册

> 10 平台内容一键发布工具（微信/知乎/微博/抖音/小红书/视频号/快手/今日头条/YouTube/TikTok）

---

## 1. 安装

1. 前往 [GitHub Releases](https://github.com/Colinchiu007/Multi-Publish/releases) 下载最新版
2. Windows: 运行 `Multi-Publish.Setup.x.x.x.exe` 安装
3. 安装完成后桌面出现 `Multi-Publish` 图标，双击启动

---

## 2. 首次启动

启动后 App 自动完成：
- ✅ 检测 Python 环境
- ✅ 安装 Python 依赖
- ✅ 安装 Playwright 浏览器
- ✅ 启动后端服务

**注意：** 首次启动可能需要 1-2 分钟，取决于网络速度。

---

## 3. 添加账号

**步骤：**
1. 左侧导航 → **账号管理**
2. 点击 "+ 新增账号"
3. 选择平台（如"微信公众号"）
4. 弹出浏览器窗口 → 扫码登录
5. 登录成功后自动关闭窗口
6. 账号列表中显示新账号 ✅

**支持的登录方式：**
| 平台 | 登录方式 |
|------|---------|
| 微信公众号 | 微信扫码 |
| 知乎 | 账号密码 / 手机号 |
| 微博 | 扫码 / 账号密码 |
| 抖音 | 扫码 / 手机号 |
| 小红书 | 扫码 / 账号密码 |
| 视频号 | 微信扫码 |
| 快手 | 手机号 / 账号密码 |
| 今日头条 | 扫码 / 账号密码 |
| YouTube | Google 账号 |
| TikTok | 扫码 / 账号密码 |

---

## 4. 发布内容

**图文发布：**
1. 左侧导航 → **一键发布**
2. 选择目标平台（可多选）
3. 输入标题
4. 在富文本编辑器中撰写内容（支持图片粘贴）
5. 点击 **发布**

**视频发布（抖音/视频号/快手/今日头条/YouTube/TikTok）：**
1. 选择支持视频的平台
2. 拖拽视频文件到上传区域（支持 mp4/mov/avi）
3. 填写标题和描述
4. 点击 **发布**

---

## 5. 批量发布 & 定时发布

**批量发布：**
- 选择多个平台 → 一键发布到所有选定平台

**定时发布：**
- 在发布页面设置发布时间的日期和时
- 到时间自动发布
- App 关闭后重启会恢复未执行的定时任务

---

## 6. 查看发布状态

- **统计看板** — 总发布数、各平台分布、成功率
- **发布历史** — 按时间倒序，可查看详情

---

## 7. 自动更新

App 启动 3 秒后自动检测新版本：
- 发现新版本 → 弹出更新对话框 → 点击"下载更新" → 下载完成后"立即重启安装"
- 支持断点续传

---

## 8. 常见问题

**Q: 添加账号时浏览器窗口不弹出？**
A: 检查是否被防火墙拦截，或手动关闭浏览器弹窗拦截。

**Q: 发布失败怎么办？**
A: 检查账号登录状态，账号过期时需重新登录。

**Q: 视频上传支持哪些格式？**
A: mp4、mov、avi。文件大小建议不超过 500MB。

**Q: 能否同时在多台电脑使用？**
A: 免费版限 1 台，Pro 版限 3 台，企业版不限。
