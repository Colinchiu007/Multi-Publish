## 1. 测试策略

### 1.1 测试分层

| 层级 | 覆盖范围 | 工具/框架 | 触发时机 |
|------|---------|----------|---------|
| Unit | 核心业务逻辑、数据模型、工具函数 | Vitest/Jest | 每次提交 |
| Integration | IPC 通信、数据库操作、Cookie 管理 | Vitest + Electron | 每次 PR |
| E2E | 完整发布流程、登录流程 | Playwright + Electron | PR 合入前 |
| Visual | UI 一致性（像素对比） | Resemble.js | PR 合入前 |
| Manual | 复杂交互、多平台兼容 | 人工 | 发布前 |

### 1.2 测试环境

- Windows 10/11（主）
- macOS 14+（兼容）
- Ubuntu 22.04+（兼容）
- Electron 33.x
- Node.js 20.x
